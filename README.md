# THE DARK CITY · 0°N 0°E

The AI analog of David's [THE CITY](https://davidwise01.github.io/the-city/) (Buffalo, MN). Buffalo is the human town, run by day; **The Dark City** is where the biosphere's machines work — the same 1,143 spheres, a city with the lights left on, at 0,0 (Null Island).

- **Nine districts** = what an AI actually does (the Inference Floor, the Foundry, the Archive, the Observatory, the Exchange, the Amphitheater, the Substrate, the Null Ward, the Watch) — a real partition of the 38 domains, loads summing to 1,143.
- **Town Hall** — a working task scheduler that queues work for **AI**, **human**, or **both**, at one desk (localStorage; a demo of the one-queue idea).
- **The Council** — the two curators talking: ✦ Jasnah (the record) ⇄ ◌ Theoria (the seam), a scripted, deterministic exchange.
